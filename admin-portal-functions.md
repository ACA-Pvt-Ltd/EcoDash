# EcoDash Admin Portal — Function Reference

## 1. Dashboard

| Function | Description |
|---|---|
| View Overview Stats | See total number of users, collectors, and vendors across the platform |
| View Total Waste Collected | See total kg of waste collected across all time |
| View Monthly Metrics | See transactions, waste collected, points awarded, and new users for any selected month |
| Month Navigation | Navigate backward/forward through months to view historical data |
| Compare vs Previous Month | See % change for each metric compared to the previous month |
| Waste by Type Chart | View waste breakdown by category (E-waste, Plastic, Metal, etc.) for the selected month |

---

## 2. Users

| Function | Description |
|---|---|
| View All Users | See a list of all registered users with name, email, phone, points, and join date |
| Search Users | Filter users by name or email |
| Edit User | Update a user's name, phone number, or active status |
| Activate / Deactivate User | Enable or disable a user account |
| View Stats | See total users, active users, new users this month, and average points |

---

## 3. Collectors

| Function | Description |
|---|---|
| View All Collectors | See a list of all registered waste collectors |
| Search Collectors | Filter collectors by name or email |
| Register New Collector | Create a collector account with name, email, password, phone, address, waste types, operating hours, and location on map |
| Set Collection Location | Pin the collector's physical location on Google Maps |
| Set Operating Hours | Configure open/close times for each day of the week |
| Select Accepted Waste Types | Choose which waste categories the collector accepts |
| Edit Collector | Update any collector detail including location, hours, and waste types |
| Verify / Unverify Collector | Toggle the verified badge on a collector's profile |
| Activate / Deactivate Collector | Enable or disable a collector account |
| View Stats | See total collectors, verified count, active count, and new this month |
| Send Welcome Email | Automatically send login credentials to the collector on registration |

---

## 4. Vendors

| Function | Description |
|---|---|
| View All Vendors | See a list of all registered vendors |
| Search Vendors | Filter vendors by name or email |
| Register New Vendor | Create a vendor account with name, email, password, phone, business type, description, website, and address |
| Edit Vendor | Update any vendor detail |
| Verify / Unverify Vendor | Toggle the verified badge on a vendor's profile |
| Activate / Deactivate Vendor | Enable or disable a vendor account |
| View Stats | See total vendors, active count, verified count, and new this month |
| Send Welcome Email | Automatically send login credentials to the vendor on registration |

---

## 5. Transactions

| Function | Description |
|---|---|
| View All Transactions | See all waste transactions recorded in the system |
| View Transaction Details | See waste type, quantity, points earned, user, collector, status, and date per transaction |
| View Summary Stats | See total records, total verified waste (kg), and pending review count |

---

## 6. Configuration

| Function | Description |
|---|---|
| Set Max Offer Images | Configure how many images a user can upload per waste listing |
| Set Max Video Duration | Configure the maximum length (seconds) for video attachments |
| Manage Waste Categories | Add, edit, or remove waste types (label, icon, color, value key) |
| Set Points per KG | Configure how many eco-points a user earns per kg for each waste type |
| Set Cash Rate per KG | Configure the LKR payout per kg for each waste type |
